路网文件：Manhattan_network.txt

第一行：node 数量，edge 数量
第二行至最后一行：node ID1，node ID2，road ID，road length


查询文件：query.txt

第一行至最后一行：起点 node ID，终点 node ID，出发时间


轨迹数据文件：route.txt

第一行至最后一行：node 数量，第一个 node ID，四个 boolean 数字，第二个 node ID，四个 boolean 数字，...


轨迹数据时间文件：time.txt

第一行至最后一行：edge 数量，驶出第二个 node 的时刻，驶出第三个 node 的时刻

注：edge 数量比 node 数量少 1
